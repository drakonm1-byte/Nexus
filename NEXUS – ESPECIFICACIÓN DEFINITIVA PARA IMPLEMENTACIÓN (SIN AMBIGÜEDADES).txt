1. ARQUITECTURA GENERAL (MEJORADA)
Nexus es una aplicación nativa de Windows que consta de:

Núcleo en Python (core/): orquestador, clasificadores, integraciones con Docker, Ollama, etc.

Subagentes (agents/): roles especializados que utilizan el modelo de lenguaje.

Herramientas (tools/): utilidades auxiliares.

Interfaz web incrustada con PyWebView.

Gestor de servicios (core/service_manager.py): lanza Docker Desktop y Ollama en segundo plano (sin ventanas), y los detiene al cerrar Nexus.

Bandeja del sistema (core/system_tray.py): icono en la bandeja que permite abrir/cerrar la ventana principal, ver estado y salir.

Contenedores Docker para sandbox.

Persistencia: SQLite, ChromaDB, archivos de usuario.

Novedades clave:

Al iniciar Nexus, solo aparece un icono en la bandeja. La ventana se abre solo cuando el usuario lo solicita.

Docker Desktop y Ollama se inician en modo silencioso (si no están ya en ejecución).

Al cerrar Nexus, se matan los procesos de Docker y Ollama, liberando RAM y VRAM.

Se utiliza psutil y subprocess con flags de Windows para ocultar ventanas.

    def _start_web_server(self):
        """Inicia el servidor Flask que sirve la interfaz web."""
        app = Flask(__name__, static_folder='webapp', static_url_path='')
        self.web_server = make_server('127.0.0.1', 8080, app, threaded=True)

        @app.route('/')
        def index():
            return send_from_directory('webapp', 'index.html')

        @app.route('/api/chat', methods=['POST'])
        def chat():
            data = request.get_json()
            user_message = data.get('message', '')
            loop = asyncio.new_event_loop()
            asyncio.set_event_loop(loop)
            try:
                response = loop.run_until_complete(self.process_request(user_message))
            finally:
                loop.close()
            return jsonify(response)

        @app.route('/api/projects', methods=['GET'])
        def list_projects():
            projects = self.project_mgr.list_projects()
            return jsonify(projects)

            path = request.args.get('path', str(Path.home()))
            try:
                files = self.project_mgr.list_files(path)
                return jsonify(files)
            except Exception as e:
                return jsonify({"error": str(e)}), 500

        @app.route('/api/state', methods=['GET'])
        def get_state():
            return jsonify({
                "session_id": self.current_session["id"] if self.current_session else None,
                "project": self.current_project,
                "queue_length": self.queue.size()
            })

        self.server_thread = threading.Thread(target=self.web_server.serve_forever, daemon=True)
        self.server_thread.start()

    def _get_js_api(self):
        class JSAPIBridge:
            def __init__(self, orchestrator):
                self.orchestrator = orchestrator

            def send_message(self, text):
                try:
                    loop = asyncio.new_event_loop()
                    asyncio.set_event_loop(loop)
                    result = loop.run_until_complete(self.orchestrator.process_request(text))
                    return result
                except Exception as e:
                    logger.exception("Error en JSAPIBridge.send_message")
                    return {"error": str(e)}
        return JSAPIBridge(self)

    def run(self):
        """Punto de entrada principal: lanza servicios, icono en bandeja, y espera."""
        # 1. Configuración inicial si es primera ejecución
        if self.config['app']['first_run']:
            self._first_run_setup()

        # 2. Cargar perfil
        profile_path = Path(self.config['app']['data_dir']) / 'profile.json'
        if profile_path.exists():
            with open(profile_path, 'r', encoding='utf-8') as f:
                self.profile = json.load(f)
        else:
            self.profile = {}

        # 3. Iniciar servicios externos (Docker, Ollama) en segundo plano
        self.service_manager.start_all()

        # 4. Iniciar servidor web (Flask)
        self._start_web_server()

        # 5. Iniciar icono en bandeja (en un hilo separado)
        self.system_tray = SystemTray(self)
        self.tray_thread = threading.Thread(target=self.system_tray.run, daemon=True)
        self.tray_thread.start()

        # 6. Mantener el hilo principal vivo hasta que se cierre la bandeja
        self.tray_thread.join()

    async def process_request(self, user_input: str, input_type: str = "text",
                              session_id: Optional[str] = None) -> dict:
        """Procesa una petición del usuario (versión asíncrona)."""
        try:
            # 1. Inicializar sesión
            if session_id:
                self.current_session = self.session_manager.load(session_id)
                if not self.current_session:
                    raise ValueError(f"Sesión {session_id} no encontrada")
            else:
                self.current_session = self.session_manager.create()
                session_id = self.current_session["id"]

            # 2. Si es voz, transcribir
            if input_type == "voice":
                user_input = await self.voice.transcribe(user_input)

            # 3. Obtener contexto visual
            screen_context = self.ui.get_active_window_context()

            # 4. Clasificación preliminar
            prelim = self.classifier.quick_classify(user_input, screen_context)

            # 5. Recuperar memoria episódica
            memories = self.memory.search(prelim["embedding"], limit=3)

            # 6. Adaptación lingüística
            normalized = self._adapt_language(user_input, prelim["intent"], screen_context)

            # 7. Descomposición de comandos
            tasks = self.sequence_parser.parse(normalized)

            # 8. Encolar tareas
            for task in tasks:
                task_info = {
                    "text": task,
                    "prelim_intent": prelim["intent"],
                    "priority": "high" if input_type != "background" else "normal"
                }
                self.queue.enqueue(task_info)

            # 9. Procesar la cola
            results = []
            while True:
                next_task = self.queue.next()
                if not next_task:
                    break
                if next_task["priority"] == "high":
                    self._pause_background()
                try:
                    result = await self._execute_task(next_task)
                    results.append(result)
                except Exception as e:
                    logger.exception("Task execution failed")
                    self._handle_task_failure(next_task, e)
                    continue

            # 10. Persistir sesión
            self.session_manager.save(self.current_session)
            return {"results": results, "session_id": session_id}

        except Exception as e:
            logger.critical("Unhandled exception in process_request", exc_info=True)
            return {"error": str(e), "session_id": session_id}

    async def _execute_task(self, task: dict) -> dict:
        """Ejecuta una tarea individual con el ciclo Builder-Auditor-Refinamiento."""
        context = self._get_current_context()
        intent = self.classifier.classify(task["text"], context)

        team = self._select_team(intent)

        estimated_tokens = 500 * intent["complexity"]
        provider = self.gateway.select_provider(intent, estimated_tokens)
        if not provider:
            raise RuntimeError("No hay proveedores disponibles")

        llm = LLMProvider(provider)

        builder = Builder(llm, system_prompt=self._load_prompt("builder"))
        auditor = Auditor(llm, system_prompt=self._load_prompt("auditor"))
        researcher = Researcher(llm) if "Researcher" in team else None
        architect = Architect(llm) if "Architect" in team else None
        security = SecurityExpert(llm) if "SecurityExpert" in team else None

        knowledge = self._retrieve_knowledge(intent, task["text"])

        code = None
        test_code = None
        sandbox_result = None
        for iteration in range(3):
            code = builder.generate(task["text"], knowledge)
            if not code:
                continue

            lint_report = auditor.lint(code)
            security_issues = auditor.security_scan(code)
            if security_issues:
                builder.add_constraint(security_issues)
                continue

            test_code = auditor.generate_tests(code)
            sandbox_result = self.sandbox.run(code, test_code)
            if sandbox_result["exit_code"] == 0:
                break

            diagnostic = self._diagnose_error(sandbox_result["logs"])
            builder.add_constraint(diagnostic)

        quality = self._evaluate_quality(code, intent["intent"])
        if quality["pass"]:
            self.vector_store.add_gold(code, intent, quality)
            if self.current_project:
                self._git_commit(code, self.current_project)

        self.undo_manager.record("code_generation", code)
        self.telemetry.record_task(intent["intent"], quality["pass"], duration=0)

        return {
            "code": code,
            "quality": quality,
            "logs": sandbox_result.get("logs") if sandbox_result else "",
            "lint": lint_report
        }

    # Métodos auxiliares (implementación real)
    def _get_current_context(self) -> dict:
        return {
            "window": self.ui.get_active_window_context(),
            "last_memory": self.memory.get_last(),
            "project": self.current_project

    def _select_team(self, intent: dict) -> list:
        from core.agent_team_loader import load_team_config
        team_config = load_team_config("config/agent_team.yaml")
        complexity = intent["complexity"]
        resources = intent["resources"]
        for team in team_config["teams"]:
            if team["complexity_min"] <= complexity <= team["complexity_max"]:
                agents = team["agents"].copy()
                if "conditional" in team and team["conditional"]["if"] in resources:
                    agents.extend(team["conditional"]["add"])
                return agents
        return ["Builder"]

    def _load_prompt(self, name: str) -> str:
        prompt_path = Path("config/prompts") / f"{name}.txt"
        if prompt_path.exists():
            return prompt_path.read_text(encoding="utf-8")
        return "Eres un asistente útil."

    def _retrieve_knowledge(self, intent: dict, query: str) -> str:
        results = self.vector_store.search(query, top_k=3)
        return "\n".join([doc["text"] for doc in results])

    def _adapt_language(self, text: str, intent: str, context: dict) -> str:
        from core.language_adapter import LanguageAdapter
        adapter = LanguageAdapter()
        return adapter.normalize(text)

    def _pause_background(self):
        pass

    def _handle_task_failure(self, task: dict, exception: Exception):
        logger.error("Task %s failed: %s", task, exception)

    def _diagnose_error(self, logs: str) -> str:
        if "ImportError" in logs:
            return "<ENVIRONMENT_LAW> La librería faltante debe ser instalada. Usa pip install.</ENVIRONMENT_LAW>"
        elif "SyntaxError" in logs:
            return "<ENVIRONMENT_LAW> Error de sintaxis. Corrige la línea indicada.</ENVIRONMENT_LAW>"
        else:
            return "<ENVIRONMENT_LAW> Error desconocido. Revisa los logs.</ENVIRONMENT_LAW>"

    def _evaluate_quality(self, code: str, intent: str) -> dict:
        from tools.linter import run_linters
        result = run_linters(code)
        profile = self.config["quality_profiles"].get(
            self.config["development"]["default_quality_profile"], {})
        pass_criteria = (result["cyclomatic_complexity"] <= profile.get("cyclomatic_complexity", 10) and
                         result["pylint_errors"] == 0)
        return {"pass": pass_criteria, "metrics": result}

    def _git_commit(self, code: str, project_path: str):
        from tools.git_manager import GitManager
        git = GitManager(project_path)
        git.commit("Nexus: código generado")

    def shutdown(self):
        """Limpieza completa al salir."""
        logger.info("Cerrando Nexus...")
        if self.web_server:
            self.web_server.shutdown()
        if self.window:
            self.window.destroy()
        self.service_manager.shutdown()
        self.queue.save(Path(self.config['app']['data_dir']) / 'queue.json')
        self.undo_manager.save(Path(self.config['app']['data_dir']) / 'undo.log')
        logger.info("Nexus cerrado correctamente.")
3.2. Gestor de servicios (core/service_manager.py)
python
"""
Gestiona procesos externos (Docker, Ollama, etc.) lanzándolos en segundo plano
y asegurando su terminación al cerrar Nexus.
"""

import subprocess
import psutil
import os
import logging
import time
from typing import Optional

logger = logging.getLogger(__name__)

class ServiceManager:
    """Controla servicios externos: inicio silencioso, monitoreo, terminación."""

    def __init__(self, config: dict):
        self.config = config
        self.processes = {}  # nombre -> objeto subprocess.Popen

    def start_all(self):
        """Inicia todos los servicios configurados que no estén ya en ejecución."""
        for name, svc in self.config.get("services", {}).items():
            self.start_service(name)

    def start_service(self, name: str):
        """Inicia un servicio específico si no está corriendo."""
        svc = self.config["services"].get(name)
        if not svc:
            logger.warning(f"Servicio {name} no configurado")
            return
        if self._is_running(svc["process_name"]):
            logger.info(f"{name} ya está en ejecución")
            return
        executable = os.path.expandvars(svc["executable"])
        if not os.path.exists(executable):
            logger.error(f"Ejecutable {executable} no encontrado")
            return
        try:
            startupinfo = subprocess.STARTUPINFO()
            startupinfo.dwFlags |= subprocess.STARTF_USESHOWWINDOW
            startupinfo.wShowWindow = subprocess.SW_HIDE
            process = subprocess.Popen(
                [executable],
                startupinfo=startupinfo,
                creationflags=subprocess.CREATE_NO_WINDOW,
                stdout=subprocess.DEVNULL,
                stderr=subprocess.DEVNULL
            )
            self.processes[name] = process
            logger.info(f"{name} iniciado en segundo plano (PID {process.pid})")
            time.sleep(3)  # dar tiempo a que arranque
        except Exception as e:
            logger.exception(f"Error al iniciar {name}: {e}")

    def _is_running(self, process_name: str) -> bool:
        for proc in psutil.process_iter(['name']):
            if proc.info['name'] == process_name:
                return True
        return False

    def stop_all(self):
        """Detiene todos los servicios que este gestor inició y mata huérfanos."""
        for name, proc in self.processes.items():
            self._terminate_process(proc)
        self._kill_orphans()

    def _terminate_process(self, proc: subprocess.Popen):
        try:
            parent = psutil.Process(proc.pid)
            children = parent.children(recursive=True)
            for child in children:
                child.terminate()
            parent.terminate()
            gone, alive = psutil.wait_procs(children + [parent], timeout=5)
            for p in alive:
                p.kill()
            logger.info(f"Proceso {proc.pid} terminado")
        except psutil.NoSuchProcess:
            pass

    def _kill_orphans(self):
        """Mata procesos que puedan haber quedado sueltos (Docker, Ollama)."""
        target_names = ['Docker Desktop.exe', 'ollama.exe', 'com.docker.service', 'dockerd.exe']
        for proc in psutil.process_iter(['name', 'pid']):
            if proc.info['name'] in target_names:
                try:
                    proc.terminate()
                    logger.info(f"Proceso huérfano {proc.info['name']} (PID {proc.info['pid']}) terminado")
                except:
                    pass

    def shutdown(self):
        """Método de limpieza final."""
        self.stop_all()

class IntentClassifier:
    INTENTS = ["software_dev", "system", "file_management", "automation", "personal_assistant"]

    def __init__(self):
        self.encoder = SentenceTransformer('all-MiniLM-L6-v2')
        examples = {
            "software_dev": "write code program develop software create application",
            "system": "start stop shutdown reboot configure settings",
            "file_management": "copy move delete organize files rename folder",
            "automation": "automatic schedule repeat every day when then",
            "personal_assistant": "reminder calendar weather email note"
        }
        self.intent_embeddings = {k: self.encoder.encode(v) for k, v in examples.items()}

    def quick_classify(self, text: str, context: dict = None) -> dict:
        emb = self.encoder.encode(text)
        similarities = {}
        for intent, emb_ex in self.intent_embeddings.items():
            sim = np.dot(emb, emb_ex) / (np.linalg.norm(emb) * np.linalg.norm(emb_ex))
            similarities[intent] = float(sim)
        best = max(similarities, key=similarities.get)
        return {"intent": best, "embedding": emb.tolist(), "confidence": similarities[best]}

    def classify(self, text: str, context: dict = None) -> dict:
        emb = self.encoder.encode(text)
        similarities = {}
        for intent, emb_ex in self.intent_embeddings.items():
            sim = np.dot(emb, emb_ex) / (np.linalg.norm(emb) * np.linalg.norm(emb_ex))
            similarities[intent] = float(sim)
        best = max(similarities, key=similarities.get)

        complexity = min(10, len(text.split()) // 10 + 1)

        resources = []
        if "gpu" in text.lower() or "cuda" in text.lower():
            resources.append("gpu")
        if "network" in text.lower() or "api" in text.lower() or "internet" in text.lower():
            resources.append("network")

        if context and context.get("window_title"):
            if "Explorer" in context["window_title"] and best != "file_management":
                best = "file_management"
            elif "Code" in context["window_title"] or "IDE" in context["window_title"]:
                best = "software_dev"

        return {
            "intent": best,
            "complexity": complexity,
            "resources": resources,
            "confidence": similarities[best]
        }

class QuotaManager:
    def __init__(self, config_path: str):
        self.config_path = config_path
        self._lock = threading.RLock()
        self._load()

    def _load(self):
        with open(self.config_path, "r", encoding="utf-8") as f:
            self.data = yaml.safe_load(f)
        self.providers = self.data["providers"]
        for p in self.providers:
            if p["type"] == "cloud":
                if p["limits"]["type"] == "credit" and "remaining_credit" not in p["limits"]:
                    p["limits"]["remaining_credit"] = p["limits"]["initial_credit"]
                elif p["limits"]["type"] == "quota" and "used_today" not in p["limits"]:
                    p["limits"]["used_today"] = 0
                elif p["limits"]["type"] == "free_tier" and "used_this_month" not in p["limits"]:
                    p["limits"]["used_this_month"] = 0

    def _save(self):
        with self._lock:
            with open(self.config_path, "w", encoding="utf-8") as f:
                yaml.dump(self.data, f, allow_unicode=True)

    def has_quota(self, provider: dict, estimated_tokens: int) -> bool:
        if provider["type"] == "local":
            return True
        limits = provider["limits"]
        if limits["type"] == "credit":
            return limits.get("remaining_credit", 0) > 0
        elif limits["type"] == "quota":
            return limits.get("used_today", 0) < limits.get("requests_per_day", 0)
        elif limits["type"] == "free_tier":
            return limits.get("used_this_month", 0) + estimated_tokens < limits.get("tokens_per_month", 0)
        return False

    def consume(self, provider: dict, tokens_used: int):
        if provider["type"] == "local":
            return
        limits = provider["limits"]
        if limits["type"] == "credit":
            cost = (tokens_used / 1000) * (provider["cost_per_1k_tokens"]["input"] + provider["cost_per_1k_tokens"]["output"])
            limits["remaining_credit"] -= cost
        elif limits["type"] == "quota":
            limits["used_today"] += 1
        elif limits["type"] == "free_tier":
            limits["used_this_month"] += tokens_used
        self._save()

    def reset_quotas(self):
        now = time.localtime()
        for p in self.providers:
            if p["type"] != "cloud":
                continue
            limits = p["limits"]
            if limits["type"] == "quota" and "reset_time" in limits:
                reset_hour = int(limits["reset_time"].split(":")[0])
                if now.tm_hour == reset_hour and now.tm_min == 0:
                    limits["used_today"] = 0
            elif limits["type"] == "free_tier" and "reset_day" in limits:
                if now.tm_mday == limits["reset_day"] and now.tm_hour == 0:
                    limits["used_this_month"] = 0
        self._save()

class APIGateway:
    def __init__(self, quota_manager: QuotaManager):
        self.quota = quota_manager

    def select_provider(self, intent: dict, estimated_tokens: int = 1000) -> dict:
        available = []
        for p in self.quota.providers:
            if self.quota.has_quota(p, estimated_tokens):
                available.append(p)
        if not available:
            return None
        available.sort(key=lambda x: (x["type"] != "cloud", -self._get_remaining(x)))
        return available[0]

    def _get_remaining(self, provider: dict) -> float:
        if provider["type"] == "local":
            return 0
        limits = provider["limits"]
        if limits["type"] == "credit":
            return limits.get("remaining_credit", 0)
        elif limits["type"] == "quota":
            return limits.get("requests_per_day", 0) - limits.get("used_today", 0)
        elif limits["type"] == "free_tier":
            return limits.get("tokens_per_month", 0) - limits.get("used_this_month", 0)
        return 0

        self.network_disabled = network_disabled

    def run(self, code: str, test_code: str = None, timeout: int = 30) -> dict:
        with tempfile.TemporaryDirectory() as tmpdir:
            tmpdir = os.path.abspath(tmpdir)
            code_path = os.path.join(tmpdir, "main.py")
            with open(code_path, "w", encoding="utf-8") as f:
                f.write(code)
            volumes = {tmpdir: {"bind": "/workspace", "mode": "rw"}}

            if test_code:
                test_path = os.path.join(tmpdir, "test_suite.py")
                with open(test_path, "w", encoding="utf-8") as f:
                    f.write(test_code)
                command = "pytest test_suite.py"

                container = self.client.containers.run(
                    self.image,
                    command,
                    volumes=volumes,
                    working_dir="/workspace",
                    mem_limit=self.memory_limit,
                    nano_cpus=int(self.cpu_limit * 1e9),
                    network_disabled=self.network_disabled,
                    detach=True,
                    remove=False

                logs = container.logs(stdout=True, stderr=True).decode('utf-8')
                container.remove()
                return {"exit_code": result["StatusCode"], "logs": logs}

                "INSERT INTO sessions (id, state, created_at, updated_at) VALUES (?, ?, ?, ?)",

            conn.execute(

        dest = os.path.join(os.path.expanduser("~"), self.onedrive_folder, "NexusBackup")
        try:
            shutil.copytree(source, dest, dirs_exist_ok=True)
            logger.info("Sincronización completada")
        except Exception as e:
            logger.error("Error en sincronización: %s", e)

    def resolve_conflict(self, local_version, remote_version):
        if self.conflict_policy == "auto_use_newest":
            return local_version if local_version["timestamp"] > remote_version["timestamp"] else remote_version
        elif self.conflict_policy == "auto_merge":
            return {**local_version, **remote_version}
        else:
            return {"local": local_version, "remote": remote_version}

import re

class PrivacyFilter:
    def anonymize(self, code: str) -> str:
        identifiers = re.findall(r'\b[a-zA-Z_][a-zA-Z0-9_]*\b', code)
        mapping = {}
        for i, ident in enumerate(identifiers):
            if ident not in mapping and ident not in {'def', 'class', 'if', 'else', 'for', 'while', 'return'}:
                mapping[ident] = f"var_{i}"
        for orig, new in mapping.items():
            code = re.sub(rf'\b{orig}\b', new, code)
        return code

from pathlib import Path
import os

class ProjectManager:
    def __init__(self, base_dir: str):
        self.base_dir = Path(base_dir)
        self.base_dir.mkdir(parents=True, exist_ok=True)

    def create_project(self, name: str) -> Path:
        project_path = self.base_dir / name
        if project_path.exists():
            raise FileExistsError(f"El proyecto {name} ya existe")
        project_path.mkdir()
        (project_path / "src").mkdir()
        (project_path / "tests").mkdir()
        (project_path / "docs").mkdir()
        return project_path

    def get_file_tree(self, project_path: Path) -> dict:
        tree = {"name": project_path.name, "children": []}
        for item in project_path.iterdir():
            if item.is_dir():
                tree["children"].append(self.get_file_tree(item))
            else:
                tree["children"].append({"name": item.name, "type": "file"})
        return tree

    def list_projects(self) -> list:
        return [p.name for p in self.base_dir.iterdir() if p.is_dir()]

    def list_files(self, path: str) -> list:
        p = Path(path)
        if not p.exists():
            raise FileNotFoundError(f"Ruta no encontrada: {path}")
        result = []
        for item in p.iterdir():
            result.append({"name": item.name, "is_dir": item.is_dir()})
        return result

import os
import requests
import hashlib

class OfflineCache:
    def __init__(self, cache_dir: str):
        self.cache_dir = cache_dir
        os.makedirs(cache_dir, exist_ok=True)

    def get(self, url: str) -> str:
        cache_key = hashlib.sha256(url.encode()).hexdigest()
        cache_path = os.path.join(self.cache_dir, cache_key)
        if os.path.exists(cache_path):
            with open(cache_path, "r", encoding="utf-8") as f:
                return f.read()
        else:
            response = requests.get(url)
            if response.status_code == 200:
                with open(cache_path, "w", encoding="utf-8") as f:
                    f.write(response.text)
                return response.text
        return ""

    def refresh(self, url: str):
        self.get(url)

import subprocess
import tempfile

import ast

class Auditor:
    def __init__(self, llm_provider, system_prompt: str, temperature: float = 0.1):
        self.llm = llm_provider
        self.system_prompt = system_prompt
        self.temperature = temperature

    def lint(self, code: str) -> dict:
        with tempfile.NamedTemporaryFile(mode="w", suffix=".py", delete=False) as f:
            f.write(code)
            tmp = f.name
        try:
            pylint = subprocess.run(["pylint", "--score=n", tmp], capture_output=True, text=True)
            mypy = subprocess.run(["mypy", "--ignore-missing-imports", tmp], capture_output=True, text=True)
            bandit = subprocess.run(["bandit", "-r", tmp], capture_output=True, text=True)
            return {"pylint": pylint.stdout, "mypy": mypy.stdout, "bandit": bandit.stdout}
        finally:
            os.unlink(tmp)

    def security_scan(self, code: str) -> list:
        issues = []
        try:
            tree = ast.parse(code)
            for node in ast.walk(tree):
                if isinstance(node, ast.Call):
                    if isinstance(node.func, ast.Name) and node.func.id in ["eval", "exec", "compile"]:
                        issues.append(f"Línea {node.lineno}: uso de {node.func.id}")
                    elif isinstance(node.func, ast.Attribute) and node.func.attr in ["system", "popen"]:
                        issues.append(f"Línea {node.lineno}: posible llamada a os.system")
        except SyntaxError as e:
            issues.append(f"SyntaxError: {e}")
        return issues

    def generate_tests(self, code: str) -> str:
        prompt = f"""Código:
{code}

Genera un test completo con pytest que cubra la funcionalidad principal.
Responde solo con el código Python, sin explicaciones."""

                    {"role": "user", "content": prompt}]
        response = self.llm.chat_completion(messages, temperature=self.temperature)
        tests = response.content

def scan_secrets(code: str) -> list:
    patterns = [
        (r'(api_key|apikey|secret|token)\s*=\s*["\']([^"\']+)["\']', "API Key"),
        (r'password\s*=\s*["\']([^"\']+)["\']', "Password"),
    ]
    found = []
    for pattern, name in patterns:
        matches = re.findall(pattern, code, re.IGNORECASE)
        for match in matches:
            found.append(f"{name} detectada: {match[1]}")
    return found

import time
import psutil

def load_test(func, iterations=10):
    cpu_samples = []
    mem_samples = []
    for _ in range(iterations):
        start_cpu = psutil.cpu_percent(interval=0.1)
        start_mem = psutil.virtual_memory().used
        func()
        end_cpu = psutil.cpu_percent(interval=0.1)
        end_mem = psutil.virtual_memory().used
        cpu_samples.append(end_cpu - start_cpu)
        mem_samples.append(end_mem - start_mem)
    return {"avg_cpu": sum(cpu_samples)/iterations, "avg_mem": sum(mem_samples)/iterations}

6.3. Construcción del instalador
Para crear un ejecutable único con PyInstaller:

bash
pip install pyinstaller
pyinstaller --onefile --windowed --add-data "config;config" --add-data "webapp;webapp" --add-data "prompts;prompts" --add-data "assets;assets" --name "Nexus" main.py
El ejecutable se generará en dist/Nexus.exe. Se puede distribuir como un único archivo.

6.4. Ejecución como servicio de Windows
Usar NSSM (Non‑Sucking Service Manager):

bash
nssm install Nexus "C:\path\to\nexus\dist\Nexus.exe"
nssm start Nexus

NOTAS FINALES DEL EXPERTO
Seguridad en la terminación de procesos: Se utiliza psutil para matar procesos hijos. Es necesario ejecutar Nexus con privilegios de administrador para poder terminar Docker Desktop correctamente. Si no se tienen, se puede configurar Docker para que se ejecute como servicio y detenerlo con net stop. Se recomienda documentar esto al usuario.

Rutas de ejecutables: Las rutas en settings.yaml usan variables de entorno (%LOCALAPPDATA%). Se expanden con os.path.expandvars() en service_manager.py.

Resiliencia: Si Docker u Ollama no están instalados, Nexus debe continuar funcionando (aunque el sandbox o el LLM local no estén disponibles). Se deben capturar las excepciones y mostrar mensajes de advertencia.

Pruebas: Incluir pruebas que verifiquen que los servicios se inician y detienen correctamente, usando mocks para subprocess y psutil.

Compatibilidad con otros SO: Si se desea portar a Linux/macOS, service_manager.py deberá usar comandos como systemctl y rutas diferentes.

Con esta especificación, el equipo de desarrollo tiene todos los elementos para construir Nexus de manera profesional y sin ambigüedades. Cualquier duda adicional, estoy a disposición.
