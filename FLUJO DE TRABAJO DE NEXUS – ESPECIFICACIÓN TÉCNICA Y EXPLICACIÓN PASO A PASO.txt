0. INTRODUCCIÓN: ¿QUÉ ES NEXUS?
Para todos
Nexus es un asistente que vive en tu ordenador. Puedes hablarle o escribirle como si fuera una persona. Él te ayuda a hacer tareas como:

Escribir programas (software) que tú le pidas.

Organizar tus archivos, carpetas, fotos.

Controlar aplicaciones (abrir, cerrar, mover ventanas).

Recordarte cosas, gestionar tu calendario.

Aprender a hacer cosas nuevas si le enseñas.

Es completamente gratuito y funciona sin conexión a internet (aunque si tienes internet puede usar servicios gratuitos para ser más rápido). Es accesible: puedes usar la voz, escribir, o incluso hacer gestos con los dedos si tienes pantalla táctil.

Para programadores
Nexus es una aplicación nativa de Windows (empaquetada como .msi o .exe) que integra:

Un orquestador escrito en Python (el núcleo), que se comunica mediante PyWebView para la interfaz web. PyWebView incrusta un navegador (Chromium/Edge/WebKit2) directamente en la ventana de la aplicación, eliminando la sobrecarga de un proceso separado de Electron.

Módulos de reconocimiento de voz (local con Vosk o Windows Speech), síntesis de voz (Windows TTS), control de aplicaciones (UI Automation), gestión de archivos, sandbox Docker (para ejecutar código de forma segura), y proveedores de IA (locales con Ollama o APIs en la nube con capas gratuitas).

Gestor de servicios externos (ServiceManager) que lanza Docker Desktop y Ollama en segundo plano (sin ventanas) y los detiene al cerrar Nexus.

Bandeja del sistema (SystemTray) que muestra un icono en la bandeja de Windows, desde donde el usuario puede abrir la ventana principal, ver el estado o salir. La ventana de PyWebView se abre solo cuando el usuario lo solicita.

Persistencia de estado en SQLite y sistema de archivos, con copias de seguridad automáticas en OneDrive (opcional).

Arquitectura modular que permite añadir habilidades, plugins y extensiones sin reiniciar.

"""
Gestiona procesos externos (Docker, Ollama, etc.) lanzándolos en segundo plano
y asegurando su terminación al cerrar Nexus.
"""

import subprocess
import psutil
import os
import logging
import time
from typing import Optional

logger = logging.getLogger(__name__)

class ServiceManager:
    def __init__(self, config: dict):
        self.config = config
        self.processes = {}  # nombre -> objeto subprocess.Popen

    def start_all(self):
        for name, svc in self.config.get("services", {}).items():
            self.start_service(name)

    def start_service(self, name: str):
        svc = self.config["services"].get(name)
        if not svc:
            logger.warning(f"Servicio {name} no configurado")
            return
        if self._is_running(svc["process_name"]):
            logger.info(f"{name} ya está en ejecución")
            return
        executable = os.path.expandvars(svc["executable"])
        if not os.path.exists(executable):
            logger.error(f"Ejecutable {executable} no encontrado")
            return
        try:
            startupinfo = subprocess.STARTUPINFO()
            startupinfo.dwFlags |= subprocess.STARTF_USESHOWWINDOW
            startupinfo.wShowWindow = subprocess.SW_HIDE
            process = subprocess.Popen(
                [executable],
                startupinfo=startupinfo,
                creationflags=subprocess.CREATE_NO_WINDOW,
                stdout=subprocess.DEVNULL,
                stderr=subprocess.DEVNULL
            )
            self.processes[name] = process
            logger.info(f"{name} iniciado en segundo plano (PID {process.pid})")
            time.sleep(3)  # dar tiempo a que arranque
        except Exception as e:
            logger.exception(f"Error al iniciar {name}: {e}")

    def _is_running(self, process_name: str) -> bool:
        for proc in psutil.process_iter(['name']):
            if proc.info['name'] == process_name:
                return True
        return False

    def stop_all(self):
        for name, proc in self.processes.items():
            self._terminate_process(proc)
        self._kill_orphans()

    def _terminate_process(self, proc: subprocess.Popen):
        try:
            parent = psutil.Process(proc.pid)
            children = parent.children(recursive=True)
            for child in children:
                child.terminate()
            parent.terminate()
            gone, alive = psutil.wait_procs(children + [parent], timeout=5)
            for p in alive:
                p.kill()
            logger.info(f"Proceso {proc.pid} terminado")
        except psutil.NoSuchProcess:
            pass

    def _kill_orphans(self):
        target_names = ['Docker Desktop.exe', 'ollama.exe', 'com.docker.service', 'dockerd.exe']
        for proc in psutil.process_iter(['name', 'pid']):
            if proc.info['name'] in target_names:
                try:
                    proc.terminate()
                    logger.info(f"Proceso huérfano {proc.info['name']} (PID {proc.info['pid']}) terminado")
                except:
                    pass

    def restart_service(self, name: str):
        self.stop_service(name)
        self.start_service(name)

    def shutdown(self):
        self.stop_all()

"""
Icono en la bandeja del sistema (system tray) con menú contextual.
Permite abrir/cerrar la ventana principal, ver estado y salir.
"""

import sys
import threading
from pathlib import Path
import logging

logger = logging.getLogger(__name__)

class SystemTray:
    def __init__(self, orchestrator, icon_path: str = None):
        self.orchestrator = orchestrator
        self.icon = None
        self.icon_path = icon_path or self._default_icon()
        self._running = False

    def _default_icon(self) -> str:
        icon_file = Path(__file__).parent.parent / "assets" / "nexus_icon.ico"
        return str(icon_file) if icon_file.exists() else None

    def run(self):
        import pystray
        from PIL import Image

        if self.icon_path and Path(self.icon_path).exists():
            image = Image.open(self.icon_path)
        else:
            image = Image.new('RGB', (64, 64), color='blue')

        menu = pystray.Menu(
            pystray.MenuItem("Abrir Nexus", self._on_open),
            pystray.MenuItem("Estado", self._on_status),
            pystray.MenuItem("Salir", self._on_quit)
        )

        self.icon = pystray.Icon("nexus", image, "Nexus Assistant", menu)
        self._running = True
        self.icon.run()

    def _on_open(self, icon, item):
        self.orchestrator.show_window()

    def _on_status(self, icon, item):
        import win32api
        win32api.MessageBox(0, "Nexus está funcionando correctamente.\n\nServicios activos:\n- Docker\n- Ollama", "Estado", 0x40)

    def _on_quit(self, icon, item):
        self.orchestrator.shutdown()
        if self.icon:
            self.icon.stop()
        sys.exit(0)

    def stop(self):
        if self.icon:
            self.icon.stop()
        self._running = False
